# MCP Server Playground

이 저장소는 **Python 기반 MCP(Model Context Protocol) 서버를 개발하고 실험**하기 위한 레포지토리입니다.

## 목적

- MCP 서버의 기본 구조 설계
- 도구(Tool), 리소스(Resource), 프롬프트(Prompt) 인터페이스 구현
- 로컬 환경에서 빠르게 테스트 가능한 개발 워크플로우 구축

## 이 레포지토리에서 다루는 범위

- MCP 서버 애플리케이션 코드
- 서버 기능 테스트용 샘플 구성
- 실행/디버깅을 위한 최소한의 문서화

## 빠른 시작 (예정)

아래 항목은 서버 코드가 추가되면 순차적으로 채워집니다.

1. Python 버전 및 의존성 설치
2. MCP 서버 실행
3. 클라이언트 연결 테스트

## 개발 방향

- 작게 시작해서 점진적으로 기능 확장
- 재사용 가능한 모듈 구조 유지
- 테스트 가능한 단위로 기능 분리

## 로드맵

- [ ] 기본 MCP 서버 엔트리포인트 구성
- [ ] 샘플 Tool/Resource/Prompt 추가
- [ ] 로컬 테스트 시나리오 문서화
